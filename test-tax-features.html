测试税点功能

测试步骤

  1. 验证单价显示顺序是否正确：
    • 单价（按片，含运费）
    • 单价（按平方，含运费）
    • 单价（按片，不含运费）
    • 单价（按平方，不含运费）
  2. 测试税点功能：
    • 输入瓷砖尺寸和数量
    • 输入进货成本
    • 设置税点为13%
    • 点击"计算价格"按钮
    • 验证税额是否正确计算（税额 = (进货成本 + 运费 + 利润) × 税点）
    • 验证总价是否包含税额
  3. 测试损耗率默认值：
    • 重置计算器
    • 验证损耗率默认值是否为0%
  4. 测试保存和加载记录功能：
    • 设置税点并完成计算
    • 保存记录
    • 在历史记录中查看该记录
    • 验证税额是否正确显示
    • 点击"加载到计算器"按钮
    • 验证税点是否正确加载

预期结果

  • 单价显示顺序应符合要求
  • 税点默认值应为0%
  • 损耗率默认值应为0%
  • 税额计算应正确：税额 = (进货成本 + 运费 + 利润) × 税点
  • 总价应包含税额：总价 = 进货成本 + 运费 + 利润 + 税额
  • 税额应在计算结果中显示（仅当税点大于0时）
  • 税点应正确保存和加载

计算公式

总运费 = 按重量运费 + 一口价运费 + 运费附加费

利润 = 进货成本 × 利润率

税额 = (进货成本 + 运费 + 利润) × 税点

总价 = 进货成本 + 运费 + 利润 + 税额

单价（含运费） = 总价 ÷ 数量/面积

单价（不含运费） = (进货成本 + 利润) ÷ 数量/面积
