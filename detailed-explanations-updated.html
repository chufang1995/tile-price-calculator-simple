计算结果详细说明（更新版）

详细解释瓷砖价格计算工具的各项计算结果

最新更新说明

总价计算逻辑已更新：

总价 = 进货总成本 × (1 + 利润率) + 运费

基础数据

瓷砖尺寸

瓷砖尺寸是计算的基础，包括长度、宽度和厚度。

  • 长度和宽度支持毫米(mm)和米(m)两种单位，可以通过单位切换按钮进行切换
  • 厚度固定为毫米(mm)单位
  • 系统会根据尺寸计算单砖面积和总重量

数量/面积

您可以选择按数量或面积进行计算：

  • 如果输入数量，系统会自动计算总面积
  • 如果输入面积，系统会自动计算所需瓷砖数量（向上取整）
  • 系统会根据损耗率调整计算结果

损耗率

损耗率是指在瓷砖安装过程中的预计损耗比例，默认为0%。

  • 如果设置了损耗率，系统会在计算总面积或总数量时自动加上损耗部分
  • 例如：如果需要100片瓷砖，损耗率为5%，则系统会计算为105片

成本计算

进货总成本

进货总成本是根据您输入的进货价格计算的瓷砖采购成本。

  • 如果只输入了按平方的进货价：进货总成本 = 总面积 × 进货价（元/㎡）
  • 如果只输入了按片的进货价：进货总成本 = 总数量 × 进货价（元/片）
  • 如果同时输入了两种价格，系统会自动选择更划算的方式：进货总成本 = min(总面积 × 进货价（元/㎡）, 总数量 × 进货价（元/片）)

运费计算

运费是指瓷砖的运输成本，有两种计算方式：

  • 一口价运费：直接输入固定的运费金额
  • 按重量运费：根据瓷砖总重量和每吨运费计算
  • 如果同时设置了两种运费，系统会优先使用一口价运费
  • 运费附加费会添加到运费总额中

合计总成本

合计总成本是进货总成本加上运费（不包含附加费）。

  • 使用一口价运费时：合计总成本 = 进货总成本 + 一口价运费（不包含附加费）
  • 使用按重量运费时：合计总成本 = 进货总成本 + 按重量计算的运费

价格计算（更新重点）

利润计算

利润是根据进货总成本和利润率计算的收益。

  • 利润 = 进货总成本 × 利润率（%）
  • 利润率是您期望的收益率，以百分比表示

总价计算（新逻辑）

总价 = 进货总成本 × (1 + 利润率) + 运费

根据最新更新，总价的计算逻辑已调整：

  • 首先计算进货总成本的利润：进货总成本 × 利润率
  • 然后加上进货总成本得到含利润的成本：进货总成本 × (1 + 利润率)
  • 最后加上运费得到总价：进货总成本 × (1 + 利润率) + 运费

税额计算

税额是根据总价（税前）和税点计算的税费。

  • 税额 = 总价（税前） × 税点（%）
  • 最终总价 = 总价（税前） + 税额

单价计算

单价是指每片瓷砖或每平方米瓷砖的价格。

  • 含运费单价（片） = 总价 ÷ 总数量
  • 不含运费单价（片） = (进货总成本 + 利润) ÷ 总数量
  • 含运费单价（㎡） = 总价 ÷ 总面积
  • 不含运费单价（㎡） = (进货总成本 + 利润) ÷ 总面积

示例说明

示例计算

输入参数：

  • 瓷砖尺寸：800mm × 800mm × 10mm
  • 数量：100片
  • 进货价（按平方）：100元/㎡
  • 一口价运费：500元
  • 运费附加费：100元
  • 利润率：20%
  • 税点：0%

计算过程：

  • 单砖面积：0.8m × 0.8m = 0.64㎡
  • 总面积：0.64㎡ × 100片 = 64㎡
  • 进货总成本：64㎡ × 100元/㎡ = 6400元
  • 运费总额：500元 + 100元 = 600元
  • 总价：6400元 × (1 + 20%) + 600元 = 7680元 + 600元 = 8280元
  • 利润：8280元 - 6400元 - 600元 = 1280元
  • 含运费单价（片）：8280元 ÷ 100片 = 82.80元/片
  • 不含运费单价（片）：(6400元 + 1280元) ÷ 100片 = 76.80元/片
