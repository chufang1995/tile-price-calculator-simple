最终计算逻辑

当前所有计算逻辑已更新完毕，最终的计算流程如下：

  1. 利润 = 进货总成本 × 利润率 + 运费附加费
  2. 总价（不含税运） = 进货总成本 × (1 + 利润率)
  3. 税前总价 = 总价（不含税运） + 运费
  4. 总价 = 税前总价 × (1 + 税点) = (总价（不含税运） + 运费) × (1 + 税点)

计算逻辑总结

经过多次调整，瓷砖价格计算工具的计算逻辑现已完全符合要求。以下是最终的计算逻辑总结：

1. 基础计算

  • 单片面积 = 长度 × 宽度
  • 总面积 = 单片面积 × 数量 × (1 + 损耗率)
  • 总数量 = 面积 ÷ 单片面积 × (1 + 损耗率)（向上取整）
  • 总重量 = 单砖重量 × 总数量

2. 成本计算

  • 进货总成本 = min(总面积 × 进货价（元/㎡）, 总数量 × 进货价（元/片）)
  • 运费 = 一口价运费 + 运费附加费 或 总重量（吨） × 运费单价（元/吨）
  • 合计总成本 = 进货总成本 + 一口价运费（不含附加费） 或 进货总成本 + 按重量计算的运费

3. 价格计算

  • 利润 = 进货总成本 × 利润率 + 运费附加费
  • 总价（不含税运） = 进货总成本 × (1 + 利润率)
  • 税前总价 = 总价（不含税运） + 运费
  • 总价 = 税前总价 × (1 + 税点) = (总价（不含税运） + 运费) × (1 + 税点)

4. 单价计算

  • 含运费单价（片） = 总价 ÷ 总数量
  • 不含运费单价（片） = 总价（不含税运） ÷ 总数量
  • 含运费单价（㎡） = 总价 ÷ 总面积
  • 不含运费单价（㎡） = 总价（不含税运） ÷ 总面积
