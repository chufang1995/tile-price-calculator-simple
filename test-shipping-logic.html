测试运费计算逻辑

测试步骤

  1. 测试按重量计算运费：
    • 输入瓷砖尺寸：300×600×10毫米
    • 输入数量：100片
    • 输入单片重量：5公斤
    • 输入按重量运费单价：100元/吨
    • 不输入一口价运费
    • 点击"计算价格"按钮
    • 验证运费是否为50元（100片 × 5公斤/片 ÷ 1000 × 100元/吨）
  2. 测试一口价运费：
    • 输入瓷砖尺寸：300×600×10毫米
    • 输入数量：100片
    • 输入单片重量：5公斤
    • 输入按重量运费单价：100元/吨
    • 输入一口价运费：80元
    • 点击"计算价格"按钮
    • 验证运费是否为80元（一口价优先）
  3. 测试运费附加费：
    • 输入瓷砖尺寸：300×600×10毫米
    • 输入数量：100片
    • 输入一口价运费：100元
    • 输入运费附加费：20元
    • 点击"计算价格"按钮
    • 验证运费是否为120元（100元 + 20元）
  4. 测试合计总成本计算：
    • 输入瓷砖尺寸：300×600×10毫米
    • 输入数量：100片
    • 输入进货成本：1000元
    • 输入一口价运费：100元
    • 点击"计算价格"按钮
    • 验证合计总成本是否为1100元（1000元 + 100元）

预期结果

  • 当只设置按重量运费时，运费 = 总重量（吨） × 运费单价（元/吨）
  • 当同时设置按重量运费和一口价运费时，优先使用一口价运费
  • 运费附加费始终会添加到运费中：最终运费 = 基础运费 + 附加费
  • 合计总成本 = 进货总成本 + 最终运费

计算公式

基础运费 = 按重量运费 或 一口价运费（优先使用一口价）

最终运费 = 基础运费 + 运费附加费

合计总成本 = 进货总成本 + 最终运费

利润 = 进货总成本 × 利润率

税额 = (进货总成本 + 最终运费 + 利润) × 税点

总价 = 进货总成本 + 最终运费 + 利润 + 税额
