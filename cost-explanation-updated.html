合计总成本计算说明（更新版）

详细解释合计总成本的计算逻辑

更新后的计算逻辑

合计总成本 = 进货总成本 + 一口价运费（不含附加费） 或 合计总成本 = 进货总成本 + （重量 × 运费单价/吨）

根据最新的要求，合计总成本的计算逻辑已更新。主要变化是：当使用一口价运费时，运费附加费不再计入合计总成本。

1. 进货总成本计算

进货总成本根据您选择的计算方式不同而不同：

  • 按平方计算：进货总成本 = 总面积 × 进货价（元/㎡）
  • 按片计算：进货总成本 = 总数量 × 进货价（元/片）
  • 如果同时设置了两种价格，则系统会自动选择更划算的方式

2. 运费计算

运费有两种计算方式，系统会根据您的设置自动选择：

  • 一口价运费：运费 = 一口价运费 + 运费附加费
  • 按重量运费：运费 = 总重量（吨） × 运费单价（元/吨）
  • 如果同时设置了两种运费，则系统会优先使用一口价运费

3. 合计总成本计算（更新重点）

合计总成本的计算逻辑已更新：

  • 使用一口价运费时：
    合计总成本 = 进货总成本 + 一口价运费（不包含运费附加费）
  • 使用按重量运费时：
    合计总成本 = 进货总成本 + 按重量计算的运费

4. 利润和总价计算

利润和总价的计算逻辑保持不变：

  • 利润 = 进货总成本 × 利润率（%）
  • 总价 = 合计总成本 + 利润

示例说明

示例1：使用一口价运费

输入参数：

  • 瓷砖尺寸：800mm × 800mm × 10mm
  • 数量：100片
  • 进货价（按平方）：100元/㎡
  • 一口价运费：500元
  • 运费附加费：100元
  • 利润率：20%

计算过程：

  • 总面积：64㎡
  • 进货总成本：64㎡ × 100元/㎡ = 6400元
  • 运费总额：500元 + 100元 = 600元
  • 合计总成本：6400元 + 500元 = 6900元（不含附加费）
  • 利润：6400元 × 20% = 1280元
  • 总价：6900元 + 1280元 = 8180元

示例2：使用按重量运费

输入参数：

  • 瓷砖尺寸：800mm × 800mm × 10mm
  • 数量：100片
  • 进货价（按平方）：100元/㎡
  • 单砖重量：20kg
  • 按重量运费：100元/吨
  • 利润率：20%

计算过程：

  • 总面积：64㎡
  • 进货总成本：64㎡ × 100元/㎡ = 6400元
  • 总重量：100片 × 20kg = 2000kg = 2吨
  • 运费：2吨 × 100元/吨 = 200元
  • 合计总成本：6400元 + 200元 = 6600元
  • 利润：6400元 × 20% = 1280元
  • 总价：6600元 + 1280元 = 7880元
