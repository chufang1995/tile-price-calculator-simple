测试新功能

测试步骤

  1. 验证标题在手机上是否显示在一行
  2. 测试一口价运费功能：
    • 输入瓷砖尺寸和数量
    • 不输入单片重量和按重量运费
    • 输入一口价运费（如100元）
    • 点击"计算价格"按钮
    • 验证运费是否正确显示
  3. 测试运费附加费功能：
    • 输入瓷砖尺寸和数量
    • 输入进货成本
    • 输入运费附加费（如50元）
    • 点击"计算价格"按钮
    • 验证附加费是否计入利润
  4. 测试组合运费功能：
    • 输入瓷砖尺寸、数量和单片重量
    • 输入按重量运费单价
    • 输入一口价运费
    • 输入运费附加费
    • 点击"计算价格"按钮
    • 验证总运费是否为按重量运费+一口价运费
    • 验证附加费是否计入利润
  5. 测试保存和加载记录功能：
    • 完成计算后保存记录
    • 在历史记录中查看该记录
    • 点击"加载到计算器"按钮
    • 验证所有运费相关字段是否正确加载

预期结果

  • 标题在手机上应显示在一行
  • 一口价运费应正确添加到总运费中
  • 运费附加费应计入利润计算
  • 总运费 = 按重量运费 + 一口价运费
  • 利润 = 进货成本 × 利润率 + 运费附加费
  • 所有新添加的字段应正确保存和加载

计算公式

总运费 = 按重量运费 + 一口价运费 + 运费附加费

利润 = 进货成本 × 利润率

总价 = 进货成本 + 总运费 + 利润

单价（不含运费） = (进货成本 + 利润) ÷ 数量/面积

单价（含运费） = 总价 ÷ 数量/面积
